📖 Dokumentasi API Komik

API untuk mendapatkan komik terbaru.

📌 Endpoint

https://apikomik.vercel.app/api

Mengambil daftar komik populer.

Example Response:

              {
                "success": true,
                "message": "Berhasil mengambil data",
                "data": [
                  {
                    "link": "",
                    "title": "",
                    "variant": "",
                    "type": "",
                    "chapter": "",
                    "last_update": "",
                    "img": ""
                  }
                ]
              }

Mengambil daftar komik terbaru berdasarkan halaman.

Parameters:

  • page (integer, opsional) → Nomor halaman yang ingin diambil.

Example Response:

              {
                "success": true,
                "message": "Berhasil mengambil data",
                "data": {
                  "current_page": "",
                  "total_page": "",
                  "link": "",
                  "title": "",
                  "variant": "",
                  "type": "",
                  "chapter": "",
                  "last_update": "",
                  "img": ""
                }
              }

Mengambil detail komik berdasarkan link yang diberikan.

Parameters:

  • link (string, wajib) → Link unik dari komik yang ingin diambil datanya.

Example Response:

              {
                "success": true,
                "message": "Berhasil mengambil data",
                "data": {
                "title": "",
                  "first_chapter": {
                    "title": "",
                    "chapter_url": "",
                  },
                  "last_chapter": {
                    "title": "",
                    "chapter_url": "",
                  },
                    "img": "",
                    "ratting": "",
                    "alternative_title": "",
                    "status": "",
                    "author": "",
                    "ilustrator": "",
                    "Type": "",
                    "tema": "",
                    "genre": "",
                    "short_sinopsis": "",
                    "sinopsis": "",
                    "related": [{
                      "link": "",
                      "img": "",
                      "title": "",
                      "sinopsis": "",
                      "variant": "",
                      "Type": "",
                    }],
                    "chapter_list": [
                    {
                      "link": "",
                      "chapter": "",
                      "update": ""
                    }]
                  }
              }

Mendapatkan konten bacaan dari sebuah komik berdasarkan link yang diberikan.

Parameters:

  • link (string, wajib) → Parameter ini adalah bagian unik dari URL komik yang digunakan untuk mengambil kontennya. Link ini bisa berupa slug atau ID spesifik dari komik yang ingin dibaca.

Example Response:

              {
                "success": true,
                "message": "Berhasil mengambil data",
                "data": {
                  "title": "",
                  "prev_chapter": "",
                  "daftar_chapter": "",
                  "next_chapter":"",
                  "content": "",
                }
              }

Endpoint ini digunakan untuk mencari komik berdasarkan judul yang diberikan. API akan mengembalikan daftar komik yang sesuai dengan pencarian.

Parameters:

  • title (string, wajib) → Judul atau kata kunci dari komik yang ingin dicari.

Example Response:

              {
                "success": true,
                "message": "Berhasil mengambil data",
                "data": [
                  "link": "",
                  "title": "",
                  "variant": "",
                  "type": "",
                  "chapter": "",
                  "last_update": "",
                  "img": ""
                ]
              }
